Description

Medicine Alert is very helpful application to make the list of medicines with time alert. User could input the name of medicine, add description, choose alert's time intervals and moment of start. The App could be use by everyone but is especially prepared for mature people - big buttons and text fields make the App easier to use.

What users can do:

- Define as many medicines as they need.
- Make a timetable, which remind them about medicines, at the right moment.
- They can also add a description about each of these medicines.

How To Import Files to Eclipse

 

1. Unzip package into your workspace folder

2. Open Eclipse. (More information about Elipse download, installation and configuration you can find HERE). We recommend use Eclipse Kepler.

3. Import project to Eclipse:

  • File->Import

  • Choose Android-> "Existing Android Code Into Workspace"

  • Browse file folder in your workspace

  • Click "Finish" to import

4. Import Google Play Services library and Android Support Library same way

5. Click rihgt mouse button on project and select Properies->Android

6. Click "Add..." to add both libraries
